## Задача 1

- абстракция;
- инкапсуляция;
- наследование;
- полиморфизм.


### Пример оформления результата с использованием кода:

```swift
protocol Flying {
    func fly(from: String, to: String)
}

protocol Carrier {
    func load()
    func refuel()
}

class Bird: Flying {
    func fly(from: String, to: String) { ... }
}

class Airplane: Flying, Carrier {

    func fly(from: String, to: String) { ... }
    func load() { ... }
    func refuel() { ... }
}

```
